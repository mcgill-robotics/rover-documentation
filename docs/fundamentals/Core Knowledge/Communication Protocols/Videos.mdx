# Embedding YouTube Videos in Docusaurus

You can easily embed YouTube videos into your Docusaurus pages using the `iframe` HTML tag. Here's how:

## Steps:

1.  **Get the YouTube Video Embed Code:**
    * Go to the YouTube video you want to embed.
    * Click the "Share" button below the video.
    * Select the "Embed" option.
    * Copy the `<iframe>` code provided.

2.  **Paste the Embed Code into your MDX File:**
    * In your Docusaurus MDX file, paste the copied `<iframe>` code where you want the video to appear.

Here's an example of how to embed a YouTube video:
